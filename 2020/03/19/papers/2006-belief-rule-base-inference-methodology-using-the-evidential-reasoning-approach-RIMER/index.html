2006-Belief Rule-Base Inference Methodology Using the Evidential Reasoning Approach—RIMER

发表于 2020-03-19 更新于 2020-05-01 分类于 papers
本文字数： 15k 阅读时长 ≈ 13 分钟

Yang J B , Liu J , Wang J , et al. Belief rule-base inference methodology using the evidential reasoning Approach-RIMER[J]. IEEE Transactions on Systems Man & Cybernetics Part A Systems & Humans, 2006, 36(2):266-285.

译文：2006-Belief Rule-Base Inference Methodology Using the Evidential Reasoning Approach—RIMER

在许多用于知识表示的替代方法中，规则似乎是表达各种类型知识的最常见形式之一。在规则库系统中，有两个基本组件：知识库和推理引擎。 将它们结合起来，可以根据专家建立的规则和用户提供的观察事实推断出有用的结论。

文中报告了基于D–S证据理论，决策理论和模糊集理论的基于混合规则的系统的设计和实现的研究，提出一种新的方法，用于使用置信结构对混合规则库进行建模，并使用证据推理（ER）方法在规则库系统中进行推理。该方法被称为使用证据推理的通用规则库推理方法（RIMER）。

在已建立的置信规则库中，每个前件的输入都转换为该前件的参考值的分布。此分布描述了每个前件被激活的程度。此外，if-then规则的前提条件构成一个整体属性，称为分组前提属性。规则的激活权重可以通过聚合规则中所有前提条件被激活的程度来生成。在这种情况下，if-then规则可以被认为是在规则具有一定程度的置信度的情况下，将一个分组的前提属性评估为一个输出项的评估问题。最后，使用ER方法实现规则库系统的推理。
