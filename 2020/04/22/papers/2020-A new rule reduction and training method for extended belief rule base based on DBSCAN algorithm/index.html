2020-A new rule reduction and training method for extended belief rule base based on DBSCAN algorithm

发表于 2020-04-22 更新于 2020-05-01 分类于 papers
本文字数： 33k 阅读时长 ≈ 30 分钟

摘要

规则约简是置信规则库系统众多研究的重点之一，在某些情况下，过多的冗余规则可能是规则库系统的关注点。尽管规则约简方法已在置信规则库系统中广泛使用，但扩展置信规则库系统（置信规则库系统的扩展）仍然缺少在扩展置信规则库（EBRB）系统中约简和训练规则的方法。为此，本文提出了一种EBRB约简和训练方法。基于基于密度的噪声应用空间聚类（DBSCAN）算法，提出了一种新的EBRB约简方法，该方法将访问EBRB中的所有规则，并融合融合阈值范围内的规则。此外，还提出了使用参数学习的EBRB训练方法，该方法使用一组训练数据来训练EBRB的参数，以提高EBRB系统的准确性。使用回归和分类的两个案例研究来说明所提出的EBRB约简和训练方法的可行性和有效性。比较结果表明，该方法可以有效减小EBRB的尺寸，提高EBRB系统的精度。

Keywords: Extended belief rule base, Rule reduction, Parameter learning

1. 介绍

在各种知识表示方案中，规则库系统由于具有在同一框架下表达各种类型知识的优势而被公认为最常用的框架之一[1]。它是由人类以IF-THEN规则的形式构造的，已成为决策支持系统和人工智能领域中增长最快的方法之一[2,3]。通常，使用简单的IF-THEN规则（例如“如果失败率高，那么安全性评估不佳”）来构建规则库，这是规则库系统的重要组成部分，并且在其优化方面已经吸引了众多研究[4–6]。在这些研究中，规则库约简一直是关注的焦点，因为过多的规则可能会降低规则库系统的准确性并增加其复杂性[7-9]。此外，由于IF-THEN规则的前件项和后件项都可以100％确定，尽管这种严格的知识表示方案可能无法有效地表达具有不确定性的信息，但已经开发了许多新规则库系统来增强在不确定性下处理定性和定量信息的能力。

在这一点上，Yang等[10]在2006年提出了使用证据推理（RIMER）方法的置信规则库的方法，该方法是在Dempster-Shafer证据理论[11-13]，贝叶斯概率理论[14]，模糊集和传统的基于IF-THEN规则的系统[1,17]的基础上开发的理论[15,16]。这种方法在不确定性条件下（如模糊，无知和不完整）在反映信息方面具有优势，已广泛用于消费者偏好预测[18–20]，医学诊断[21–23]，风险分析[24– 27]，以及许多其他领域。与信任函数相比[13,17]，置信规则库系统更适合涉及先验知识的问题，例如回归和分类问题。通过将置信规则分组到一个置信规则库（BRB）中，置信规则库系统可以有效地解决不同的决策问题，但是，因为BRB是置信规则库系统的重要组成部分，并且对推理结果具有重大影响，其优化方法已吸引了广泛的研究[28-30]。此外，置信规则库的约简也已成为研究人员关注的焦点，许多方法已被应用来提供期望的置信规则库约简方法[31-34]。此外，一些研究者研究了结合置信规则库约简的参数学习方法，以优化信念规则库并同时缩小其规模[35]。最近，基于RIMER方法，Liu等人[36]通过将置信结构引入每个规则的前提属性（称为扩展信念规则库（EBRB）），提出了一种更通用的置信规则库系统，以更好地处理前提属性中的不确定性。通过将置信度嵌入每个规则的整个前提属性中，EBRB可以更有效地处理各种不确定性并提供更准确的结果，并已用于健康估计[37]，传感器配置优化[38]，活动识别[39]和其他一些领域[40,41]。

但是，EBRB系统仍需要解决几个挑战。 第一个挑战是减小EBRB系统的大小并避免不一致性，第二个挑战是增强其表示能力，这对于规则库系统至关重要。

对于第一个挑战，一些研究人员通过调整EBRB系统中激活规则的数量来解决它，这可以减少推理时间[42-44]。 Calzada等[43]提出了一种用于EBRB系统的动态规则激活（DRA）方法，其中动态选择激活的规则以追求不完整和不一致之间的平衡。 Lin等[44]提出了一种基于VP-tree和MVP-tree的规则激活方法，其中在计算每个规则的激活权重时，将仅检索和访问部分规则。Yang等[45]提出了基于一致性分析的规则激活（CABRA）方法，通过使用一致的规则集作为激活框架来重新定义合适的激活规则，这使EBRB系统能够克服不一致和不完整的问题。Yang等[41]构造了扩展置信规则的激活区域，并修改了激活权重的计算公式以重新定义激活规则，并提出了新的激活规则确定和权重计算程序。但是，这些规则激活方法无法有效减小EBRB的大小，并且由于对给定输入仅激活了有限的规则，因此可能会丢失信息。关于EBRB的约简，很少有研究，Yang等[46]提出了一种基于数据包络分析（DEA）的EBRB规则约简方法，其中DEA用于评估每个规则的效率，并从EBRB中删除无效规则以减小其大小。如Yang等[46]所述，对于解决了第一个挑战，EBRB约简比规则激活更重要和有效，因为：（1）如果没有EBRB约简，EBRB的大小可能会很大，因为输入输出数据对可以很容易地转换成EBRB的置信规则； （2）对任何输入数据激活EBRB中的所有规则是造成不一致的确切原因，并且随着EBRB的大小增加，情况可能会变得更糟； （3）为每个输入数据简单激活某些规则可能会导致信息丢失，因为一些相关规则可能会被忽略。

对于第二个挑战，最常见的方法之一是增加规则库中的规则数量，这也可能增加其时间成本。 最近，规则库的训练已被用作提高置信规则库系统的推理性能的有效方法。 杨等[47]提出了一种新的基于灵敏度分析的激活权重计算和参数优化方法，通过使用新的参数优化方法，提高了EBRB系统的精度。 尽管EBRB使用训练数据来生成规则，但仍然需要训练过程来增加其表示能力和增强其性能。

为了解决上述挑战，本文将基于密度的噪声应用空间聚类（DBSCAN）算法和参数学习应用于EBRB的约简和训练。对于第一个挑战，提出了一种新的EBRB约简方法，该方法将基于不同规则之间的欧几里得距离，使用DBSCAN算法搜索并融合相似的置信规则。对于第二个挑战，使用扩展参数学习方法来训练EBRB的参数，即前提属性的置信度，后件置信度，属性权重和规则权重，以提高其准确性。对于较大的EBRB，将首先使用EBRB约简方法来缩减置信规则，然后将EBRB训练方法用于训练新构造的EBRB的参数。提出的方法的主要优点是它可以有效地减小EBRB的大小，同时保持相对较高的精度。为了说明所提方法的有效性，应用了两种情况，包括函数逼近和分类。准确性和规则数量这两个主要方面用于与其他EBRB系统进行比较。

本文的其余部分安排如下。 第2节简述了证据推理（ER）方法和EBRB系统的基础。 第3节提出了EBRB训练方法和EBRB约简方法。 第4节提供了两个案例研究，以证明所提出的EBRB约简和训练方法的效率，本文在第5节中进行了总结。

3.2 EBRB结构分析

作为BRB系统的扩展，EBRB系统可以用作通用逼近器，它可以以可接受的精度逼近数据集上的任何实连续函数。 但是，EBRB中的扩展置信规则的数量可能会对EBRB系统的准确性产生巨大影响。 如果规则数量低于阈值（即，前提属性参考等级 JJJ 的数量），那么规则数量的减少将直接导致EBRB系统准确性的降低。 但是，EBRB中规则数量的增加并不一定会导致EBRB系统准确性的提高，在某些情况下，EBRB中过多的规则可能会导致计算时间增加甚至减少EBRB系统准确性。 为了说明EBRB中的规则数量对其推理性能的影响，我们研究了以下示例。

3.3 基于DBSCAN算法的EBRB约简方法

如示例3所示，当EBRB中规则的规则权重为0时，这意味着该规则在确定EBRB系统的输出时将不起作用，因此可以被认为是多余的。 此外，如示例4所示，当EBRB中的规则过多时，过多的信息可能会对其推理性能产生负面影响，并且要花费大量时间和计算资源来计算每个规则的激活权重并获得最终结果。 在许多情况下，有些置信规则在EBRB中表示相似的知识，可以被认为是多余的规则。 因此，有必要在EBRB中搜索并融合相似的置信规则，以减小其规模并提高其效率。

在上述情况下，只有少数扩展的置信规则是单个规则，而大多数扩展的置信规则彼此相似。 使用Ester等人[52]提出的DBSCAN算法，可以通过计算它们的距离来确定相似的规则，并可以提出基于DBSCAN算法的EBRB约简方法。提出的EBRB约简方法的过程概括为算法1。

3.4 EBRB约简和训练过程

分别定义了EBRB训练方法和EBRB约简方法后，EBRB约简和训练过程如图9所示，并分三个步骤总结如下：

步骤1：初始EBRB生成

为了构建EBRB系统，需要使用Liu等人[36]提出的EBRB生成方法来构建初始EBRB，以及使用一组来自系统观察或基于专家知识的输入输出数据对作为EBRB生成过程中的训练集。 因此，可以获得具有过多置信规则的初始EBRB。

步骤2：EBRB规则约简
在使用一组训练数据生成初始EBRB之后，可能会有足够相似的置信规则，可以被视为表示相同信息的冗余规则。 因此，使用第3.3节中提出的EBRB约简方法，将搜索并融合初始EBRB中的冗余规则，从而减小了EBRB的大小。

步骤3：约简的EBRB的参数学习
EBRB约简后，应对新获得的EBRB进行训练，然后再用于进行推理过程。 因此，第3.1节介绍的参数学习方法可以应用于约简的EBRB，以增强其性能而不会丢失信息。

5. 总结

提出了一种新的EBRB约简和训练方法，以解决超大型EBRB的过多置信规则会增加计算时间和影响推理性能的问题。 本文的主要结论可以概括如下：

（1）研究了在相同数量的训练数据下具有不同数量的置信规则的EBRB系统的推理性能。 本文揭示了置信规则的数量对EBRB系统的推理性能的影响，并得出结论认为，超大型EBRB可能对EBRB系统的推理性能产生负面影响。

（2）基于DBSCAN算法，提出了EBRB约简方法。 在提出的方法中，冗余规则被搜索和融合以减少超大型EBRB中的规则数量。 使用此方法，可以融合融合阈值范围内的相似置信规则，从而有效减小EBRB的大小。

（3）先前的研究仅使用训练数据来生成EBRB，并移除了EBRB训练过程，这大大降低了EBRB系统的准确性。 因此，本文提出了一种使用参数学习的EBRB训练方法，该方法使用相同的训练数据来训练EBRB系统的参数，包括前提属性置信度，结果置信度，属性权重和规则权重，并被证明能够提高EBRB系统的准确性。

此外，通过回归和分类两个案例研究来说明该方法的有效性，结果表明该方法可以有效地减小EBRB的大小并提高其推理性能。 总之，实验证明了该方法的可行性和有效性。

在以后的研究中，我们将研究如何提高不完整EBRB的推理性能，我们相信规则添加方法是保证不完全EBRB完整性和有效性的有效方法。 此外，在EBRB约简过程中，融合阈值对于结果很重要，并且还将进一步研究如何适当选择融合阈值。

而且，如本文所示，由于未严格定义相似性度量和距离度量，因此可以进一步研究原始EBRB系统的设计。 尽管已经提出了几种修改方案，但仍将考虑对原始EBRB系统进行修改的进一步研究。
