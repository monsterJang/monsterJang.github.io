7-4 Review of Programming Contest Rules

思路

首先要知道怎么样才算最优：第一要最大化解题数 solved ，其次要最小化 total time

然后就可以去 N! 枚举解题次序，注意到题目要求 If there are several solutions, you must output the smallest sequence. ，因此枚举时的顺序要注意一下

7-5 Werewolf

思路

狼人杀游戏，N 个玩家，M 个狼人，L 个说谎者，其中说谎者中包含 [1, M-1] 个狼人

枚举 + 剪枝，枚举第 i 个玩家是否为狼人，枚举的过程中可以加入限制条件来剪枝（但实际测的时候，题目数据少加甚至不加剪枝似乎也不是不能接受），注意题目要求 If there are more than one solution, you must output the largest solution sequence ，因此枚举时要注意顺序
