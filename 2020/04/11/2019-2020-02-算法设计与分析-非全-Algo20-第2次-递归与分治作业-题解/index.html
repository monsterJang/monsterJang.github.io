7-3 整数分解为若干项之和

思路

使用递归与分治，用一个数组记录当前整数分解式子的各项值，注意题目有严格的格式要求

void dfs(int pre, int cur, vector<int>& ans);

pre 记录前一个使用的数字（格式要求分解式子的各项值是递增的）
cur 记录当前各项值的和
ans 记录当前的各项值
